## O usuário deve:
- inserir o nome da pessoa ser pesquisada, de preferência, completo.

## O usuário pode:
- caso o usuário não saiba o nome completo, pode-se utilizar alcunhas ou apelidos para facilitar a pesquisa;
- no caso de alcunhas e apelidos apenas, o modelo tenderá a pesquisar o que for mais provável.

# Exemplos:
- Jair Messias, Jair Bolsonaro ou Bolsonaro;
- Luiz Inácio, Luiz da Silva ou Lula;
- Guilherme Boulos ou Boulos
- Trovão
