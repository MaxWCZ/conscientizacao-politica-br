# Testes compactados

## Sobre
O arquivo anexado contém apenas os resultados brutos dos testes realizados em todas as versões do prompt da Camada 0.

A organização interna segue a data de modificação dos arquivos, em ordem decrescente, de modo que os testes mais antigos (por figura) correspondem às primeiras versões do prompt.

## das versões anteriores do prompt
As versões anteriores do prompt foram preservadas apenas como registro de desenvolvimento. A versão recomendada para uso é sempre a mais recente, disponível na pasta camada-0.

- Novas sobre evolução do prompt
    - Primeiras versões: foco em estrutura básica de resposta.
    - Versões intermediárias: inclusão de validação obrigatória e classificação de fontes.
    - Versão atual: consolidação de regras, limites e padronização da saída.
